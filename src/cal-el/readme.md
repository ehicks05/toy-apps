# cal-el

A learning experiment in building a simple calendar.

Currently browser-only. Events are stored in local storage.

## Prereqs

1. node

## Getting Started

1. Clone repo.
2. To install depenencies, run `npm i`.
3. For local dev, run `npm run dev`.
